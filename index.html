This web page demonstrates the basic use of JavaScript to determine the time of day by the hour. In addition, this will display a different image, text, and color based on the current time the user is active on the page.

  • Morning (5am to 12pm)
  • Afternoon (12pm to 6pm)
  • Evening (6pm to 11pm)
  • Night (11pm to 5am)
